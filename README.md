## üöÄ Getting Started

### Prerequisites
- Node.js (v16+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vinay33333/3-JS-GAME.git
   cd 3-JS-GAME
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser at `http://localhost:5173` (or the port shown in terminal).
